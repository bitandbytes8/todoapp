# Todo App
> [x] Reactjs
>
> [x] Firebase as Database
> 
> [x] Deployment
> 

## npm upgrade 

> npm install -g npm

## Setup React App
> npx create-react-app todo_app

## Setup Firebase
> npm install -g firebase-tools (Stage 1)

> sudo npm install -g firebase-tools (Stage 2)

> npm i firebase-tools

> firebase login

> firebase init

> firebase deploy

### Start the Project 

> cd todo_app
> 
> npm start
